WHAT I'VE LEARNED IN WEB DEV SPRING 2018

It has been a wonderful semester in web development. Walking in on the first day with no web development experience whatsoever, I can confidently say that I've learned a lot and have come a long way. Through the various projects that we've done this year, I have found some website styles that reflect greatly on my own design preferences. I realized I like a very classy and clean-cut style, something that's very chic and modern. I like using big pictures to portray the message that I want to give my users instead of using lots of text. My design is constantly evolving, and I still have lots to learn. I hope to design more beautiful, complex, and user friendly websites in the future.
